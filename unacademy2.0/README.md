# Unacademy Clone

This project was created using create-react-app Express and MongoDB Atlas

## How to run app

Clone the app

`git clone https://github.com/yash-ga/Unacademy-clone.git`


  
  
 
 ### How to run frontend

`cd Unacademy-clone/unacademy2.0`

`npm install`

`npm start`
